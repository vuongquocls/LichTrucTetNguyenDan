# Lịch trực YokDon

Dự án Vite + React để xếp lịch trực. Đã chuyển component vào `src/App.jsx`.

Chạy ở máy local:

```bash
npm install
npm run dev
```

Build để deploy lên Netlify (hoặc tương tự):

```bash
npm run build
```

Netlify sẽ dùng `npm run build` và publish thư mục `dist` (xem `netlify.toml`).
